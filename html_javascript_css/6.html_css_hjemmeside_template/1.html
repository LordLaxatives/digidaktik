Skriftligt skal tekst, billeder, film eller hvad du ellers vælger at inkludere, give dine personlige, kortfattede svar på følgende spørgsmål:

  • Hvad er Internettet?
  • Hvad er en server og en klient?
  • Hvad er HTML?
  • Hvad er CSS?

Du kan udnytte at du rent faktisk skriver siden med HTML og CSS til at give eksempler - som denne linje der forandres, hvis man holder musen henover den.
